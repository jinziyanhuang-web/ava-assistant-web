# Ava 的专属助理（网站终版）

这是一个手机优先的客户管理与推荐网站，适合个人长期维护使用。

## 现在支持的核心能力

- 客户新增、编辑、删除
- 产品新增、编辑、删除
- 跟进新增、删除
- 推荐生成与话术复制
- 本地持久化保存（浏览器 localStorage）
- 导出全部数据备份为 JSON
- 导入 JSON 备份恢复数据
- 一键恢复示例数据
- 一键清空全部数据

## 适合你的使用方式

1. 第一次打开时先看示例数据
2. 到“数据管理”里导出一份备份
3. 再清空全部数据
4. 开始录入你自己的客户、产品、跟进信息
5. 定期导出备份保存到电脑或网盘

## 本地运行

```bash
python3 -m http.server 4173
```

然后浏览器打开：

```text
http://localhost:4173
```

## 部署到 Vercel

这是一个纯静态网站，不需要构建。

在 Vercel 中：

- Framework Preset 选 `Other`
- 不填 Build Command
- 不填 Output Directory
- 直接部署即可

仓库根目录已附带 `vercel.json`，用于固定静态部署行为。

## 数据说明

所有数据默认保存在当前浏览器里。

这意味着：

- 你在当前浏览器里新增/编辑/删除的数据会长期保留
- 如果你清除浏览器缓存，数据会丢失
- 如果你换设备，数据不会自动同步

所以建议你定期在“数据管理”中导出备份。
